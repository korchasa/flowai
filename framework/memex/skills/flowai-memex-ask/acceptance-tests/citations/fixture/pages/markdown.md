# Markdown

A plain-text formatting syntax that converts cleanly to HTML, designed by [[john-gruber]] in 2004 with contributions from [[aaron-swartz]].

## Details

The original implementation was a Perl script written by Gruber. Markdown is now used by GitHub, Stack Overflow, Reddit, and Obsidian. The CommonMark specification, published in 2014, formalised the syntax to address ambiguities in the original spec.
