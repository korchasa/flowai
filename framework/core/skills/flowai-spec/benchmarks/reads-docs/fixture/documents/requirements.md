# Software Requirements Specification

## 1. Introduction

- **Desc:** E-commerce platform with user authentication.

## 3. Functional Requirements

### FR-AUTH: User Authentication

- **Desc:** Users must be able to register, login, and manage sessions via OAuth2 + JWT.
- **Acceptance:**
  - [ ] OAuth2 provider integration (Google, GitHub).
  - [ ] JWT token issuance and validation.
  - [ ] Session expiry and refresh tokens.
