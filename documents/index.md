# Documentation Index

Agent-maintained navigation aggregator across all linkable artifacts (FR / SDS / NFR). Rows are normally added by `plan` on first FR touch; a `maintenance` back-fill MAY populate rows for pre-existing FR anchors in one pass (FR-DOC-INDEX). Task ↔ FR navigation lives inline in SRS as `**Tasks:**` back-pointers (FR-DOC-TASK-LINK), not here.
