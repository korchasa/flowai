## CLI COMMANDS VIA `deno task`

### Usage:
```
deno task <command> [args...]
```

### Commands:
- `check` - Run all checks (cleanup, formatting, linting, analyze, build, test)
- `test <test_id>` - Run a test by relative path.
- `dev` - Run the project in development mode
